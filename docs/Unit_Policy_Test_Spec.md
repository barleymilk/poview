# POVIEW 단위 정책 회귀 테스트 명세

## 배경

- [ ] 점수 산식 및 적합성 계산의 단위 정책(`m` ↔ `cm`)이 일관되게 유지되어야 분석/비교 신뢰도가 보장된다.
- [ ] 정책 위반이나 변환 오류 시 회귀를 조기에 검출하기 위해 필수 테스트 케이스를 명문화한다.

## 목표

- [ ] 단위 변환 정확성을 검증하는 회귀 테스트 케이스를 정의한다.
- [ ] 경계값 및 등급 경계에서의 동작을 스냅샷 테스트로 고정한다.
- [ ] 구현 시 CI 필수 항목으로 포함된다.

## 범위

- [ ] 정책: 3D 좌표/렌더링은 `m`, 점수 산식/적합성 계산 입력은 `cm`
- [ ] 대상 모듈: `selectHeightM`, 단위 변환 유틸, 점수 산식(clamp/등급 경계), deep link 파서의 `distanceToTargetM` 검증
